# Contributing

Obrigado por contribuir. Siga estas regras básicas:

- Abra um branch com nome `feat/...` ou `fix/...`.
- Faça commits pequenos e atômicos com mensagens no estilo Conventional Commits.
- Rode `pre-commit` localmente antes de submeter PR.
- Não inclua segredos em commits; use `.env.example`.
